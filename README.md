# 💳 Bank Customer Risk & Transaction Analytics System

## 🚀 Overview

This project analyzes bank customer data to identify transaction patterns, detect high-risk users, and generate useful business insights.

It uses data analysis techniques along with visualization to simulate real-world banking analytics systems.

---

## ✨ Features

### 📊 Data Analysis

* Total and average balance
* Top customers
* Transaction insights

### ⚠️ Risk Detection

* Identifies high-risk customers
* Detects suspicious transaction patterns
* Classifies users as:


  * High Risk ⚠️
  * Fraud Alert 🚨

### 📈 Visualization

* Bar chart for customer transactions
* Pie chart for risk distribution

### 📁 Output

* Saves analyzed data to CSV file

---

## 🧠 Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib

---

## 📂 Dataset Format

```csv
Name,Balance,Transactions,Withdrawals,Deposits
Aman,50000,20,30000,20000
Riya,20000,35,25000,15000
Rahul,5000,40,45000,10000
```

## ▶️ How to Run

1. Install dependencies:

```bash
pip install pandas numpy matplotlib
```

2. Run the project:

```bash
python main.py
```
